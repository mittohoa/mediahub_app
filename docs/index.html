Cài đặt APK

Bản thử nghiệm cài trực tiếp (sideload), chưa qua Google Play.

  1. Tải file APK ở nút phía trên (chọn arm64-v8a cho máy đời mới).
  2. Mở file vừa tải; Android sẽ hỏi cho phép cài từ nguồn không xác định → bật cho trình duyệt/trình quản lý file.
  3. Nhấn Cài đặt → mở app → cấp quyền Ảnh khi được hỏi.

Yêu cầu Android 6.0+ (minSdk 23). Đây là bản beta ký bằng khoá thử nghiệm — Android có thể cảnh báo; đây là điều bình thường với app cài ngoài Store. Ảnh, thẻ, khuôn mặt, vị trí đều lưu cục bộ trên máy bạn.
